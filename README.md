# Spring_WebProject
<p>
Spring MVC 패턴을 이용해 만든 기본형 웹 프로젝트입니다. 웹 개발 일련의 과정을 이해하고 검증하고자 프로젝트를 만들었으며 Spring Framework, myBatis framework, 프로젝트 관리 도구로 maven과 WAS tomcat9.0 ,  Docker 를 이용한 OracleXE DB 개발환경으로 작업하였습니다. 
</p>
